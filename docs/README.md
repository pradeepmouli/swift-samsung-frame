## SwiftSamsungFrame DocC Documentation

This directory contains the static HTML documentation generated by DocC for the SwiftSamsungFrame package.

### GitHub Pages Deployment

- The contents of this folder are ready for deployment to GitHub Pages.
- Ensure the `.nojekyll` file is present to allow files and folders starting with `_` to be served.
- Set the GitHub Pages source to the `docs/` folder on the `master` or `main` branch in your repository settings.
- The documentation will be available at:
  `https://pradeepmouli.github.io/swift-samsung-frame/` (or your fork's equivalent)

### Updating Documentation

Regenerate the documentation with DocC and overwrite the contents of this folder as needed.
